# AirMouse 🖐️

A touchless system that lets you control your computer using hand gestures in real time.

## 🚀 Features
- Cursor control using index finger
- Click using hand gestures
- Scroll with two-finger gesture
- Drag & drop with pinch
- Real-time tracking with MediaPipe

## 🛠 Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## ▶️ How to Run
```bash
python gesture_mouse.py
